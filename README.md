# Vue update watcher

The chrome extention that helps you not miss out on the recent Vue updates!

## Commands:

Project setup  
`yarn`

Compiles and hot-reloads for development  
`yarn serve`

Compiles and minifies for production  
`yarn build`

Run your unit tests  
`yarn test:unit`

Run your end-to-end tests  
`yarn test:e2e`

Lints and fixes files  
`yarn lint`
